\doxysubsection{Detailed Description}
Defines a utility class to format and send environmental sensor data as JSON via HTTP POST requests. 

This class provides a static method to simplify the process of taking sensor readings (light, temperature, humidity), formatting them into a standard JSON object, and sending this data to a specified \doxylink{class_a_p_i}{API} endpoint using the ESP32 HTTPClient library.
