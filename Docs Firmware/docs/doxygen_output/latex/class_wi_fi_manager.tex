Initiates an attempt to connect to the configured Wi\+Fi network. 

Initiates connection to the configured Wi\+Fi network implementation.

If the current status is {\ttfamily DISCONNECTED} or {\ttfamily CONNECTION\+\_\+\+LOST}, this method sets the status to {\ttfamily CONNECTING}, updates the LED, resets reconnection counters, and calls {\ttfamily Wi\+Fi.\+begin()} with the stored credentials. Does nothing if the status is already {\ttfamily CONNECTED} or {\ttfamily CONNECTING}. \begin{DoxyReturn}{Returns}
{\ttfamily true} if a connection attempt was newly initiated. Returns {\ttfamily true} if already {\ttfamily CONNECTED}.
