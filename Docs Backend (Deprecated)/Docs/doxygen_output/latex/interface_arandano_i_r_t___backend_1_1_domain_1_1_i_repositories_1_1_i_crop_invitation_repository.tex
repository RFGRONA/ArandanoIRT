\doxysubsection{Detailed Description}
Defines the repository contract specifically for managing Crop\+Invitation\+Entity instances. 

Includes operations for retrieving active invitations by code and marking them as used, in addition to the standard CRUD-\/like operations inherited from IRepository$<$\+T$>$.
