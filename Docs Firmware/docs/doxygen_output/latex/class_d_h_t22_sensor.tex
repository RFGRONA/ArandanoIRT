\doxysubsection{Detailed Description}
Wrapper class simplifying the use of a DHT22 temperature and humidity sensor. 

This class encapsulates the Adafruit DHT library object for a DHT22 sensor, providing straightforward methods to initialize the sensor and read measurements.
