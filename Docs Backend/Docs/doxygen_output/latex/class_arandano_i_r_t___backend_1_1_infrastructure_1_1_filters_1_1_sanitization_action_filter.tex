\doxysubsection{Detailed Description}
An ASP.\+NET Core action filter that automatically sanitizes string properties marked with the Sanitize\+Html\+Attribute on action method arguments (typically DTOs). 

Uses the registered ISanitizer\+Service to remove potentially harmful elements.
