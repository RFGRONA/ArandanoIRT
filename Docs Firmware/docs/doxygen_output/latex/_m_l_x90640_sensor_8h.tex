\doxysubsection{Detailed Description}
Defines a wrapper class for the MLX90640 Thermal Camera sensor. 

Provides an interface to initialize and read thermal data frames (32x24 pixels) from an MLX90640 sensor using a specific I2C bus instance and the Adafruit MLX90640 library. Also includes helper methods to calculate average, maximum, and minimum temperatures from the latest frame.
