Detailed Description

Implements IAuditEntryGenerator to create audit trail entries in the Auditsystemtable for changes made to specific "system" or related data entities like Plantstatehistory and Plantobservation.

An IEnumerable<T> collection of objects, where each object represents a distinct audit log entry to be persisted. The specific type of these objects depends on the implementation.

Generates a single Auditsystemtable record for INSERT, UPDATE, or DELETE operations on supported entity types (Plantstatehistory, Plantobservation). Performs row-level auditing ("ALL" column) using the IAuditHelperService. CropId is explicitly set to null for entries generated by this instance.
