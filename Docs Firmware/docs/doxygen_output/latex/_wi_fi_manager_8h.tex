\doxysubsection{Detailed Description}
Defines a class to manage Wi\+Fi station (STA) connection state and handle non-\/blocking reconnection. 

This class encapsulates Wi\+Fi connection logic for an ESP32 in Station mode, including\+:
\begin{DoxyItemize}
\item Storing network credentials (SSID, password).
\item Initiating connection attempts.
\item Tracking the connection status via a state machine ({\ttfamily Connection\+Status} enum).
\item Handling standard Wi\+Fi system events (connect, disconnect, got IP) via static callbacks.
\item Implementing non-\/blocking, timed reconnection attempts with configurable limits.
\item Providing a method to perform a basic check for internet connectivity. It uses an internal singleton pattern ({\ttfamily \+\_\+instance}) to allow the static event handlers (required by the ESP32 Wi\+Fi event system) to correctly update the state of the single active \doxylink{class_wi_fi_manager}{Wi\+Fi\+Manager} object. It also leverages the \doxylink{class_l_e_d_status}{LEDStatus} class for visual feedback. 
\end{DoxyItemize}
