\doxysubsection{Detailed Description}
Implements the \doxylink{class_error_logger}{Error\+Logger} utility class for sending remote error logs. 

\begin{DoxyNote}{Note}
This implementation relies on the ESP32 Wi\+Fi and HTTPClient libraries, Arduino\+Json, and requires Wi\+Fi to be connected for logs to be sent. 

It also depends on a globally instantiated {\ttfamily api} object (from \doxylink{_a_p_i_8h}{API.\+h}/\+API.cpp) to check backend status via {\ttfamily api.\+check\+Backend\+Status()} before attempting to send logs. 
\end{DoxyNote}
