\doxysubsection{Detailed Description}
Defines a utility class to send thermal data (JSON) and image data (JPEG) as a multipart/form-\/data POST request. 

Provides a static method to package thermal sensor readings (as a JSON object containing an array) and a captured camera image (as JPEG binary data) into a single HTTP POST request suitable for APIs expecting multipart/form-\/data. Uses ESP32 HTTPClient and Arduino\+Json libraries.
