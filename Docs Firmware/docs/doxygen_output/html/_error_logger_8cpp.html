Detailed Description

Implements the ErrorLogger utility class for sending remote error logs.

Note
This implementation relies on the ESP32 WiFi and HTTPClient libraries, ArduinoJson, and requires WiFi to be connected for logs to be sent.
It also depends on a globally instantiated api object (from API.h/API.cpp) to check backend status via api.checkBackendStatus() before attempting to send logs.
