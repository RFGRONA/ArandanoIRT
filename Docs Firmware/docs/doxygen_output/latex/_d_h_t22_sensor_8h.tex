\doxysubsection{Detailed Description}
Defines a wrapper class for the DHT22 temperature and humidity sensor. 

Provides a simple interface to initialize and read temperature and humidity values from a DHT22 sensor connected to a specific GPIO pin, using the Adafruit DHT sensor library.
