Generates a single {\ttfamily Auditcrop} record for INSERT, UPDATE, or DELETE operations on a {\ttfamily Crop} entity. It currently performs row-\/level auditing by setting {\ttfamily Column\+Name} to "{}\+ALL"{} and serializing the entire entity\textquotesingle{}s original/current values using the IAudit\+Helper\+Service. For INSERT operations, Record\+Id and Crop\+Id are set to placeholders (0/null) as they are not yet available.
