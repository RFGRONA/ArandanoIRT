Task< Result > ArandanoIRT_Backend.Application.Interfaces.IServices.IAuthPasswordService.RequestPasswordResetAsync ( ForgotPasswordRequestDto request)

Initiates the password reset process for a given email address.

Generates a reset token, stores it, and typically sends it via email. Protects against email enumeration attacks.

Parameters
requestDTO containing the user's email.
Returns
A Task representing the asynchronous operation. Always returns a success result to prevent email enumeration, even if the email doesn't exist or validation fails internally.

Implemented in ArandanoIRT_Backend.Application.Services.AuthPasswordService.
