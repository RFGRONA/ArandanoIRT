Detailed Description

Defines a wrapper class for the OV2640 camera module using the ESP-IDF driver.

Provides methods to initialize the camera hardware based on predefined pin configurations (specific to the project's ESP32-S3 board), capture JPEG images, and properly deinitialize the camera to release resources. Leverages the esp_camera.h functions from the ESP-IDF framework.
