\doxysubsection{Detailed Description}
Implements the ISanitizer\+Service. 

This implementation uses Html\+Sanitizer for XSS protection (default config) and then decodes HTML entities to preserve original characters. Note\+: Default Html\+Sanitizer configuration allows some safe HTML tags.
