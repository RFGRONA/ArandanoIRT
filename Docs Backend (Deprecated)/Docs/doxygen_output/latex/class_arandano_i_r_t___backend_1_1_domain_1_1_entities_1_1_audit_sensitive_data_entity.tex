\doxysubsection{Detailed Description}
Represents a single audit trail record specifically tracking actions involving access to, or operations on, sensitive data fields within specified database tables, associated with a crop. 

Note\+: This entity primarily logs the action performed (e.\+g., access, decryption) rather than the data values themselves.
