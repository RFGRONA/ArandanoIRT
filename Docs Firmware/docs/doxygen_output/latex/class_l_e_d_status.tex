\doxysubsection{Detailed Description}
Manages a single WS2812 Neo\+Pixel for displaying system status via colors. 

This class encapsulates the Adafruit\+\_\+\+Neo\+Pixel library for controlling one pixel. It maps logical system states (defined in {\ttfamily LEDState}) to specific colors.
