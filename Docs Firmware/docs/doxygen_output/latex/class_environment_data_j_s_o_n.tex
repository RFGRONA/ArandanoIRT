Constructs a JSON payload and sends environmental data to an \doxylink{class_a_p_i}{API} via HTTP POST. 

Static method implementation for constructing JSON and sending environmental data via HTTP POST.

This static method takes environmental sensor readings, formats them into a JSON object with the structure {\ttfamily \{"{}light"{}\+: value, "{}temperature"{}\+: value, "{}humidity"{}\+: value\}}, and then sends this JSON payload as the body of an HTTP POST request to the specified \doxylink{class_a_p_i}{API} URL.
