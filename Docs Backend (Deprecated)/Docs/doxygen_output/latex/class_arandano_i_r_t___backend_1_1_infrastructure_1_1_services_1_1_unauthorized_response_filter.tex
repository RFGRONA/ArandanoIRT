\doxysubsection{Detailed Description}
An Swagger operation filter that adds the 401 Unauthorized response type to operations that are protected by an Authorize attribute (identified by name). 

This filter identifies the Authorize attribute by comparing the attribute\textquotesingle{}s type name as a string.
