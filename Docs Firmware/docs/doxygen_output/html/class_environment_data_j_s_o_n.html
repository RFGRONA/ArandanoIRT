Constructs a JSON payload and sends environmental data to an API via HTTP POST.

Static method implementation for constructing JSON and sending environmental data via HTTP POST.

This static method takes environmental sensor readings, formats them into a JSON object with the structure {"light": value, "temperature": value, "humidity": value}, and then sends this JSON payload as the body of an HTTP POST request to the specified API URL.

Parameters
apiUrlThe complete URL (String) of the API endpoint that will receive the POST request. Example: "https://api.example.com/data"
lightLevelThe ambient light level reading (float, e.g., in lux).
temperatureThe temperature reading (float, e.g., in Celsius).
humidityThe relative humidity reading (float, e.g., in percentage).
Returns
true if the HTTP POST request was sent successfully AND the server responded with a success status code (HTTP 200-299). Returns false if the connection failed, the request failed to send, or the server responded with an error code.
